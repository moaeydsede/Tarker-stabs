# متابعة الأدوية (PWA موبايل)

تطبيق ويب بسيط واحترافي لمتابعة جدول الأدوية على الموبايل فقط:
- إضافة دواء (اسم/ملاحظة/مواعيد يومية أو دواء أزمة PRN)
- رفع صورة للدواء (تُحفظ داخل الجهاز)
- أزرار: **تناولت الآن** / **لم أتناوله** (يظل مستحقًا ويستمر التذكير)
- سجل تناول + تصدير CSV
- PWA: يعمل بدون إنترنت بعد فتحه مرة، ويمكن تثبيته من المتصفح

> تنبيه: التذكير بالإشعارات يعتمد على دعم النظام/المتصفح وإعدادات البطارية. أفضل تجربة تكون عند تثبيت التطبيق (PWA).

## التشغيل محليًا
افتح `index.html` مباشرة أو استخدم سيرفر محلي بسيط:
```bash
python -m http.server 8000
```
ثم افتح:
`http://localhost:8000`

## النشر على GitHub Pages
1) ارفع الملفات للمستودع (Repository)
2) من Settings -> Pages
3) اختر Branch: `main` و Folder: `/root`
4) احفظ، وسيظهر رابط النشر

## الخصوصية
كل البيانات تُحفظ في LocalStorage على جهازك فقط.


## ميزة الفترة (من تاريخ إلى تاريخ)
يمكن تحديد فترة الدواء (Start/End). لن يظهر الدواء في قائمة اليوم ولن تُرسل له تنبيهات خارج هذه الفترة.
